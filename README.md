 AI-Powered Surveillance System for Real-Time Anomaly Detection

This project presents an AI-powered surveillance system designed to detect anomalous activities in real time using advanced computer vision and deep learning techniques. 
The system leverages YOLOv8 for accurate object detection, OpenCV for video processing and frame analysis, and a Streamlit dashboard to provide an interactive 
and user-friendly interface for monitoring live surveillance feeds. By continuously analyzing video streams, the system can identify unusual events, highlight potential
security threats, and generate real-time visual alerts to assist surveillance personnel. The primary goal of this project is to improve public safety and security by providing a fast,
efficient, and intelligent monitoring solution that reduces the need for constant manual supervision while enabling timely responses to suspicious activities.
